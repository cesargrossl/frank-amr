Atualizar o ROs2
sudo apt install software-properties-common -y
sudo add-apt-repository universe
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null


instalar o ros
sudo apt update
sudo apt install ros-jazzy-desktop -y

Configurar permante

echo "source /opt/ros/jazzy/setup.bash" >> ~/.bashrc
source ~/.bashrc

Install ferramentas adicionaris

sudo apt install python3-colcon-common-extensions python3-rosdep python3-argcomplete -y
sudo rosdep init
rosdep update


asso 2: Instale Pacotes Necessários para LIDAR, SLAM e Navigation

Para RPLIDAR: Instale o pacote oficial para ROS 2 (publica dados em /scan):
apt install ros-jazzy-rplidar-ros -y

Para SLAM (mapping) e Navigation:
apt install ros-jazzy-slam-toolbox ros-jazzy-navigation2 ros-jazzy-nav2-bringup -y


Para visualização (RViz):
apt install ros-jazzy-rviz2 -y



Passo 3: Crie um Workspace ROS 2

Crie o diretório:
bashmkdir -p ~/ros2_ws/src
cd ~/ros2_ws

Copie seu código original para o workspace (para adaptação):
bashcp ~/Desktop/lidar.cpp ~/ros2_ws/src/



Passo 4: Crie um Pacote ROS 2 para o Controle do Carrinho

Crie o pacote C++:
bashcd ~/ros2_ws/src
ros2 pkg create --build-type ament_cmake carrinho_pkg --dependencies rclcpp std_msgs geometry_msgs sensor_msgs nav2_msgs pigpio

Vá para o diretório do pacote:
bashcd carrinho_pkg
